Oct. 13th – Wireframing

  • We'll talk about wireframing, it's purpose and the tools you can use to create wireframes.
  • In groups, we'll do a wireframing exercise using photoshop and Axure to better understand how wireframes can be handy tools when re-designing or designing a website
  • Wireframing Task: Imagine that you are working on a group project for class. To help everyone understand what they should be working on, you want to create a website that will allow you create manage the tasks each person group member should be working on. The website should allow users to 1) Create new tasks and edit exiting ones 2) Assign tasks to specific people 3) Mark tasks as finished 4) Group tasks together.
  • Recently, a lot of cool wireframing and prototyping tools have been developed such as Proto.io, Invision, Just in Mind, Axure, and Balsamiq. Choosing one is really a matter of finding one you like and making sure it fits your needs.
Homework – Oct. 15th Before Class
  • Start working on the wireframes for your own site using the sketches you've already created. Remember to create 3 display widths: mobile (320px), tablet (480px) and desktop (780+). Although the screen widths didn't matter in our sketches, your wireframes should be sized accurately. Here's an example of wireframes created for mobile, tablet and desktop experiences: responsive wireframes. Here's another example shown in class of a medium fidelity wireframe (notice there is some real content). For your wireframes, make sure to include as much real content as you can to get a sense of how it will actually lay out. The wireframes should grayscale (we'll worry about visual design elements such as typography and color later).
    These will be due Oct 20th. To submit, post your Invision link to the "portfolio-wireframes" slack channel.
  • Check out this article from Smashing Magazine for an introduction to media queries.
  • A cool thing: Axure, a wireframing tool that allows you to create interactive wireframes, offers free licenses to students and teachers. To get a free license, you'll need to submit proof of enrollment. Find out more about it here: Axure Education License
  • If you want a free UI kit for Photoshop (with pre-made buttons and such) for wireframing, Invision has a free one posted here: http://www.invisionapp.com/do
  • Some interactive wireframe examples shown in class: Low fidelity (Recipeas Project) and Medium fidelity (Fine Arts Library).

Oct. 22th –Design Systems & Atomic Design Theory

Objective – Consistency is important when designing for the web and has a major impact on the user's experience. Establishing conventions that users can identify is critical, especially in larger sites. Creating a design system can help with this. It can also help you plan the look and feel of your site. We'll learn about creating design systems and Brad Frost's theory of atomic design.
  • Review some styleguides and discuss what's in them. What might be useful for our portfolio projects?

  • Go over the template you guys will use to create a design system for your portfolio.
