Sept 3 - How the Internet Works + Intro to HTML, the "skeleton"

Objective – We'll talk about how the internet works, discuss some web conventions and terminology, review some basic HTML and markup our "About Me" pages.
  • Discuss what the internet is, how browsers work and the basics needed to create a webpage. If you don't already have them, please download: Chrome, Firefox & Safari.
  • Examine examples of good web design and collectively create a list of best practices in web-design. Identify common components of web pages and learn their names.
  • Discuss HTML tags <!DOCTYPE html>, <html>, <head>, <title>, <body>, <p>, <ol>, <ul>, <li>, <a>, <img>.
  • Discuss HTML attributes <img src=""> and <a href=""></a>.
  • Discuss Sublime Text 2, syntax highlighting and proper indentation.
  • Turn content from our "About Me" project into HTML.
Homework – Due before class on Sept 8th
  • Finish marking up your "About Me" pages. Make sure to include an image, a list, a paragraph and a link. Post the URL in our slack chat for credit.
  • Watch "Build your own webpage", "HTML Basics II" & "Social Networking Profile" on codecademy. Please send a screenshot showing that you've competed each of these courses.
  • Spend at least a half hour using the web inspector to see how various websites are coded. Look at the HTML and the CSS. Inspect the "About Me" page we built together, inspect this site and big name sites like Apple, Buzzfeed, Etsy, Starbucks, Amazon, Target, etc.
  • Look at the design (typeface, colors, contrast, balance) for each of Aesops fables rendered with google fonts. As you're looking, think about what typefaces you might want to use and how you would like to design your "about me" page.
  • FYI: there will be a short quiz on Sept. 15 in class on the tags we have used so far, so study up. Revisit the Codeacdemy tutorials if necessary.
  • Note: We've learned about a few HTML tags so far. There are many more available to us, some we will cover and some we will not. It's good to be aware of what tags are in the language so that you can always use the appropriate tag for the content. Check out a full list to HTML tags here. Please take note of the descriptions which tell you when and why to use each tag. Click on the tag to learn more about it and to see what attributes can be used with that tag.

Sept. 8th - Intermediate HTML + About Me Mockups

Objective – Learn some more advanced HTML tags, publish our "About Me" pages with Github, look at some other "About Me" pages and start mocking up your "About Me" pages in Photoshop.
  • Review some less common and more complicated HTML tags and when to use them. We'll look at <main>, <meta>, <main>, <table>, <div>, <form>, <nav>, <input>
  • Create Github page and commit our "About Me" pages. We'll go through this process together and talk about source control and code commits. Here's directions on how to create a page for future reference.
  • Download the Starter HTML Template. We'll use this to try out some table and form building.
  • We'll also talk about Block Level vs Inline elements and rules of use.
  • Look at some example "About Me" pages for inspiration. Jessica Hische, Alice Lee, Eduardo Marin and Katie Kovalcin all have about me pages.
  • Discuss file structure best-practices and how we'll structure our portfolio projects.
Homework – Due before class on Sept. 10th
  • Finish your "About Me" mockups and bring them to class--we'll begin building them in code! Here's an example that I created. Remember! This is just an image that will act as a guide as we create our CSS.
  • Complete these 4 courses on Codecademy: HTML Basics III, Clickable Photo Page, CSS an overview and Design a Button for Your Website. Email me screenshots of the completed course screens for each course.
  • Cool Resource Alert – If you need some dummy content, the Lorum Ipsum generator at www.lipsum.com is a great tool to quickly generate which allows you to generate blocks of gibberish latin text. I used it to create the block of gibberish used in my mockup above.
  • Cool Resource Alert Part II – If you want to add some unicode characters to your website such as ☎, ❤, or even ⛵ or ↣, check out the Unicode Character Table site, which will show you the symbol and html code you can paste into your HTML document.
