Pulp Juice Bar Branding
Role: Design

In addition to my work on the styleguide, I’m responsible for making iterative design enhancements to Priceline’s responsive post-booking experience which consists of all the confirmation pages across the various products (air, hotels, rental car and vackation packages). The overall goal for the enhancements is to create a more robust user experience by building out self-service functionality while finding ways to increase conversion without detracting from the experience. These enhancements vary in nature, scope and complexity: some slight modifications to the visual design and content hierarchy, while others are changes to the existing user flow; however, all decisions are driven by data gathered from several sources including customer service calls and A/B testing. In addition to creating the designs, I also protoype and build them in HTML and SCSS, writing production ready code which is then passed over to the developers. If the design change requires altering a pre-existing portion of the page, I directly edit the code base. This creates an efficient workflow and improves communication between developers and designers which ultimately leads to a better product. Below are a few examples of projects that I've worked on.
